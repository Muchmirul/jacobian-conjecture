# jacobian-conj

Work on the Jacobian Conjecture.

> **Jacobian Conjecture.** Let $F : \mathbb{C}^n \to \mathbb{C}^n$ be a polynomial map.
> If the Jacobian determinant $\det JF$ is a nonzero constant, then $F$ is invertible
> and its inverse is also polynomial.
